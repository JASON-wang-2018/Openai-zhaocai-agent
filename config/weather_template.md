# 天气预报推送配置

## 推送时间
- 每天 7:30

## 推送内容模板

```
🌤️ 早上好！2026年03月10日 Tuesday

【农历】正月廿二 | 惊蛰

【今日天气】
苏州地区:   
天气状态：⛅ 多云
目前温度：9.8°C   目前湿度：93%
最高气温：12.4°C 
最低气温：1.7°C
风向风级：南风1级
紫外线指数：6.5 (较高) SPF30+防晒

【穿衣建议】
❄️ 寒冷天气，建议穿羽绒服、棉服，裹严实

【生活小贴士】
🚶 适度运动，保持健康

祝您今天心情愉快！😊
```

## 字段说明

| 字段 | 数据来源 | 说明 |
|------|----------|------|
| 日期 | 系统时间 | 阳历月日+星期 |
| 农历 | 农历计算 | 正月/二月... + 初一/初二... |
| 节气 | 农历计算 | 惊蛰/清明... |
| 天气状态 | Open-Meteo API | 天气图标+文字 |
| 目前温度 | Open-Meteo API | 当前气温 |
| 湿度 | Open-Meteo API | 当前湿度百分比 |
| 最高/最低气温 | Open-Meteo API | 今日预报 |
| 风向风级 | Open-Meteo API | 风向+风力等级 |
| 紫外线指数 | Open-Meteo API | 指数+等级+建议 |
| 穿衣建议 | 温度区间判断 | 自动生成 |
| 生活小贴士 | 随机选择 | 每日一条 |

## 技术实现
- 脚本：`/home/admin/.openclaw/workspace/scripts/life_assistant.py`
- 天气API：Open-Meteo (免费无需key)
- 农历计算：内置简化算法
- 推送：飞书Webhook
