# VAVE价值分析

## 概述
VAVE (Value Analysis / Value Engineering) 通过功能定义和成本拆解，优化价值。

---

## 核心公式

```
价值(Value) = 功能(Function) / 成本(Cost)

V = F / C
```

### 提高价值的途径

| 途径 | 公式 | 说明 |
|------|------|------|
| 功能↑成本不变 | F↑/C | 增加功能 |
| 功能↑成本↓ | F↑/C↓ | 理想状态 |
| 功能不变成本↓ | F/C↓ | 降低成本 |
| 功能↑成本略↑ | F↑/C↑ | 可接受 |
| 功能↓成本↓↓ | F↓/C↓↓ | 简化版 |

## 家电产品VAVE案例

### 案例：电饭煲成本优化

**功能分析**：
| 功能 | 成本(元) | 重要性 | 价值 |
|------|----------|--------|------|
| 煮饭 | 80 | 高 | 中 |
| 保温 | 20 | 中 | 中 |
| 定时 | 15 | 中 | 低 |
| 显示 | 10 | 低 | 低 |

**改进方案**：
- 保留核心煮饭功能
- 简化保温（取消长时间保温）
- 取消LED显示，改用机械指示
- **成本降低30%**
