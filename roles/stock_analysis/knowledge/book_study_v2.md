## 一、《股市极客思考录》彭道富

### 核心
- **根基**：风险控制
- **战法**：龙头股战法
- **境界**：捻花成佛、一剑封喉

### 关键
1. 不懂风险控制 = 为他人做嫁衣
2. 利润 = 风险的解决之策
3. 形而上：投资思想；形而下：龙头股把握

## 二、《交易行为心理学》

### 四大偏差
| 偏差 | 后果 |
|------|------|
| 过度自信 | 频繁交易→年损失6.8% |
| 损失厌恶 | 亏死扛、盈早跑 |
| 羊群效应 | 接最后一棒 |
| 锚定效应 | 判断失真 |

### 对策
- 认识弱点 → 建立系统 → 严格纪律 → 持续反思

## 三、《超级短线战法》

### 三原则
- **天时**：大势判断
- **地利**：强势股+买点
- **人和**：心态纪律

### 买卖点
- **买**：圆月弯刀、突破新高、分歧转一致
- **卖**：低开冲高走、均线压制、N字无力
