# 主升+冰点反转 高胜率模型

## 核心理念
**胜率优先 > 利润最大化**
- 不追求"抓最多主升"
- 只做"最干净、最一致、最不容易失败"的结构
- 少出手 = 高胜率

---

## 提升胜率的四大核心

1. **减少出手次数** - 只做确定性机会
2. **提高过滤强度** - 多层验证
3. **加入"分歧确认"** - 不抢跑，等确认
4. **只做主线前排** - 龙头 > 跟风

---

## 主升高胜率模型 v3.0（严格过滤版）

### 第一层：市场强趋势确认（必须）

**条件**：
```
指数强趋势 :=
MA(INDEXC,20) > MA(INDEXC,60)           // 20日均线 > 60日均线
AND MA(INDEXC,20) > REF(MA(INDEXC,20),1) // 20日均线向上
AND INDEXC > MA(INDEXC,10)                // 价格在10日均线上
```

**没有这个条件**：
- ❌ 禁止新开仓
- 弱市不做强股

---

### 第二层：主线板块锁定（必须）

**条件**：
```
强于指数 := C/INDEXC > 1.05;           // 股价涨幅 > 指数涨幅5%
趋势加速 := MA(C,5) > MA(C,10) AND MA(C,10) > MA(C,20);  // 均线多头
板块共振 := 强于指数 AND 趋势加速;
```

**只做明显强势股**

---

### 第三层：龙头优先机制（强化）

**条件**：
```
近5日新高 := C >= HHV(H,5);                             // 创5日新高
涨停强度 := COUNT(C>=REF(C,1)*1.095,5)>=1;              // 5日内有涨停
龙头特征 := 近5日新高 AND 涨停强度;
```

**不是龙头结构，过滤**

---

### 第四层：量价强一致（升级版）

**条件**：
```
放量突破 := C > HHV(H,10) AND V > MA(V,10)*1.5;  // 放量突破10日高点
回调缩量 := COUNT(C<REF(C,1),3)>=1 AND V < MA(V,10)*0.7;  // 回调缩量
量价强一致 := 放量突破 AND 回调缩量;
```

**强一致 = 放量涨、缩量跌**

---

### 第五层：分歧确认买点（胜率提升关键）

**很多人死在"第一次突破"，改为只做分歧转强**：

```
分歧日 := 
  C > REF(C,1) AND C < REF(H,1)    // 冲高回落
  AND V > MA(V,10)*1.2;             // 放量

转强确认 := 
  C > REF(H,1) AND C > MA(C,5);     // 突破昨日高点

分歧转强 := 分歧日 AND REF(转强确认,1);
```

**逻辑**：
- 第一天：分歧（冲高回落）
- 第二天：转强（突破昨日高点）
- 第二天尾盘/第三天开盘买

---

### 第六层：失败压制过滤（强过滤）

**三条硬杀规则**：

```
连续3日放量滞涨 := 
  COUNT(V>MA(V,10) AND ABS(C-REF(C,1))<2, 3) >= 3;

高位加速过大 := 位置比例 > 0.8;

单日巨量阴 := 
  V > MA(V,10)*2 AND C < REF(C,1);
```

**全部必须为否**

---

### 最终高胜率选股公式

```python
选股条件 =
  指数强趋势
  AND 板块共振
  AND 龙头特征
  AND 量价强一致
  AND 分歧转强
  AND NOT(连续3日放量滞涨)
  AND NOT(高位加速过大)
  AND NOT(单日巨量阴)
```

- 换更高成功率
- = 职业化交易思维

## 冰点反转模型（待补充）

当市场处于冰点时：
- 涨停家数 < 20
- 跌停家数 > 30
- 最高连板 < 3

**反转机会**：
- 超跌反弹
- 次新股/小盘股
- 率先涨停的标杆股
