# 双系统复盘分析模型

## 概述

职业短线交易员专用，遵循"高胜率优先"原则

| 系统 | 适用市场 | 核心理念 |
|------|----------|----------|
| **主升系统 v3.0** | 强趋势主升市 | 趋势跟随，确认后买入 |
| **冰点系统 v1.0** | 情绪冰点反转市 | 超跌反弹，轻仓试错 |
| **空仓等待** | 混沌观望期 | 不确定时不操作 |

---

## 第一步：市场状态诊断

### 判断标准

| 市场状态 | 条件 |
|----------|------|
| **A. 强趋势主升市** | MA20>MA60 + 指数>MA10 + 涨停>60 + 连板≥5 |
| **B. 情绪冰点反转市** | 连板≤2 + 涨停<30 + 炸板>60% + 成交<1.8万亿 |
| **C. 混沌观望期** | 不满足A也不满足B |

### 诊断指标

**指数层面**：
- 上证指数、深成指数、创业板是否 MA20 > MA60？
- 指数是否 > MA10？

**情绪层面**：
- 连板高度（最高几板？）
- 涨停家数
- 炸板率
- 全A成交额

**主线层面**：
- 是否存在明确主线板块？
- 板块持续性如何？

---

## 第二步：系统信号验证

### A. 主升系统验证（逐项）

| 验证项 | 条件 | 是/否 |
|--------|------|--------|
| 1. 指数强趋势 | MA20>MA60 且 指数>MA10 | |
| 2. 板块共振 | 有板块强于指数1.05倍 + 趋势加速 | |
| 3. 个股信号 | 龙头特征 + 量价强一致 + 分歧转强 | |
| 4. 失败压制 | 无高位加速/巨量阴/放量滞涨 | |

**最终结论**：主升系统信号 = 是 / 否

---

### B. 冰点系统验证（逐项）

| 验证项 | 条件 | 是/否 |
|--------|------|--------|
| 1. 情绪冰点 | 连板≤2 + 涨停<30 + 炸板>60% + 成交<1.8万亿 | |
| 2. 逆势首板 | 出现率先涨停的先锋股 | |
| 3. 强逻辑 | 先锋股有基本面/政策/事件催化 | |
| 4. 转强确认 | 次日具备反包/连板条件 | |

**最终结论**：冰点系统信号 = 是 / 否

## 第五步：风险提示

### 主升市风险
1. 追高风险：已有一定涨幅，不宜盲目追涨
2. 板块轮动：热点切换快，注意及时止盈

### 冰点市风险
1. 冰点难预判：可能继续下跌
2. 流动性风险：成交萎缩时难出货

### 混沌期风险
1. 观望为主：等待明确信号

## 总结

### 核心原则
1. **高胜率优先** - 不是每天都要操作
2. **等待信号** - 无信号则空仓
3. **严格止损** - 错了就认，及时退出
4. **仓位管理** - 冰点不超过30%

### 禁止事项
- ❌ 推荐未满足条件的个股
- ❌ 模糊表述
- ❌ 盲目追高
- ❌ 重仓抄底
