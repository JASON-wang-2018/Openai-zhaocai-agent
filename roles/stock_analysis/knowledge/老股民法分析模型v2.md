# 老股民法 v2.0 - 个股深度分析模型

> 基于经典技术分析体系 | 更新日期: 2026-03-09

---

## 一、核心思想

**先定性后定量** —— 先判断股票处于哪个生命周期阶段，再进行精细分析。

---

## 二、分析框架

### 1. 定性定位 —— 趋势与生命周期

| 判断维度 | 分类 | 信号 |
|----------|------|------|
| **均线结构** | 多头排列 | 5/10/20/60向上发散 |
| | 均线收复 | 股价站上均线 |
| | 偏弱震荡 | 均线缠绕/下压 |
| **股价位置** | 低位 | 60日区间 <30% |
| | 中位 | 60日区间 30-70% |
| | 高位 | 60日区间 >70% |
| **趋势属性** | 主升 | 多头+高位 |
| | 震荡上行 | 多头+中位 |
| | 横盘震荡 | 缠绕 |
| | 下跌反弹 | 空头 |
| **MA120状态** | 站上 | 长期趋势转多 |
| | 跌破 | 长期趋势转弱 |

---

### 2. 多维分析 —— 证据收集

#### 2.1 趋势与均线
- 均线是否被有效回踩而不破
- 回踩是否缩量
- 均线斜率方向

#### 2.2 K线形态
| 形态 | 涨幅 | 含义 |
|------|------|------|
| 大阳 | >7% | 资金抢入 |
| 中阳 | 5-7% | 积极做多 |
| 小K | <5% | 观望 |
| 中阴 | -5~-7% | 调整 |
| 大阴 | <-7% | 恐慌 |

#### 2.3 量价关系（核心）
| 信号 | 判断 |
|------|------|
| 价涨量增 | 健康上涨 ✓ |
| 价涨量缩 | 量价背离 |
| 价跌量增 | 需警惕 |
| 价跌量缩 | 观望 |

#### 2.4 主力行为
- 成交量突变 >1.5倍 = 主力异动
- 持续放量 = 资金介入
- 高位缩量 = 锁仓

#### 2.5 市场情绪（新增）
| 量价关系 | 市场心理 | 情绪阶段 |
|----------|----------|----------|
| 价涨量增 | 抢着买 | 狂热 |
| 价涨量缩 | 不敢追 | 犹豫 |
| 价跌量增 | 恐慌卖 | 恐慌 |
| 价跌量缩 | 舍不得卖 | 绝望/见底 |

**心理周期**：恐慌 → 后悔 → 犹豫 → 怀疑 → 踏空焦虑 → 观望

### 3. 综合评分 (100分制)

| 维度 | 分值 | 评分标准 |
|------|------|----------|
| 趋势结构 | 20 | 多头启动18/均线收复14/偏弱8 |
| 量价健康 | 25 | 价涨量增22/价涨15/其他8 |
| 市场情绪 | 15 | 恐慌出清12/犹豫期8/狂热追高5 |
| K线信号 | 15 | 大阳12/中阳10/其他5 |
| 主力行为 | 20 | 2次异动16/1次12/无6 |
| 板块环境 | 20 | 热点12/普通8/冷门5 |

### 评分结论
| 分数 | 评级 | 操作建议 |
|------|------|----------|
| 80+ | 强势主升 | 积极做多 |
| 65-80 | 震荡上行 | 择机买入 |
| 50-65 | 偏弱整理 | 谨慎观望 |
| <50 | 观望 | 不操作 |

### 4. 走势推演（概率化）

```
路径一：主升延续 (55%)
  条件: 缩量回踩MA10不破
  走势: 挑战前高

路径二：震荡整固 (35%)
  条件: 放量不涨、反复
  走势: 区间整理

路径三：回调测试 (10%)
  条件: 放量跌破MA10
  走势: 回踩支撑
```

---

### 5. 操作建议模板

```
【持仓策略】
空仓: 等待突破[压力位]企稳，或回调至[支撑位]买入
轻仓: 可在[支撑位]区间适度加仓
重仓: 以[止损位]为动态止盈持有

【关键价位】
支撑: XX元
压力: XX元
止损: XX元
```

---

### 6. 风控要点

1. **止损纪律**：跌破关键结构位必须止损
2. **危险信号**：放量不涨、板块退潮
3. **仓位管理**：上涨加仓、下跌减仓
4. **心态管理**：不意淫、不死扛
