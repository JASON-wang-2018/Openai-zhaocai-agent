## 总结：分析方法论

### 1. 四维排序法
1. 资金行为（主力流向）
2. 涨停数量
3. 量价强度
4. 消息催化

### 2. 量价健康度
- 上涨放量 ✓
- 下跌缩量 ✓
- 板块持续放量 ✓

### 3. 三种交易机会
- 短线：龙头换手板
- 中线：主升趋势股
- 低吸：洗盘回踩

### 4. 风险信号
- 缩量 → 谨慎
- 轮动快 → 不追高
- 退潮 → 空仓
