# 股票标准化分析模式

> 基于热点题材知识库 + 曹明成庄家理论

---

## 一、分析框架

### 1. 基础数据分析
| 项目 | 说明 |
|------|------|
| 收盘价 | 当前价位 |
| 涨跌幅 | 当日表现 |
| 成交量 | 量能变化 |
| 成交额 | 资金活跃度 |

### 2. 均线系统（5层均线）
| 均线 | 周期 | 用途 |
|------|------|------|
| MA5 | 1周 | 短线支撑/压力 |
| MA10 | 2周 | 短期趋势 |
| MA20 | 1月 | 中期趋势 |
| MA60 | 3月 | 中长线 |
| MA120 | 6月 | 长线压力 |

**均线信号判断**：
- 股价站上均线 = 支撑
- 股价跌破均线 = 压力

### 3. 成交量分析
| 信号 | 判断 |
|------|------|
| 量价齐升 | 健康上涨 |
| 放量突破 | 启动信号 |
| 缩量回调 | 低吸机会 |
| 量价背离 | 风险信号 |

### 4. 主力行为识别
- **成交量突变**：放量2倍以上 = 主力异动
- **大阳线**：涨幅>7% = 资金抢入
- **低位横盘**：主力建仓
- **高位放量**：主力出货

### 5. 走势阶段（情绪周期）
| 阶段 | 特征 | 操作 |
|------|------|------|
| 启动 | 放量突破，量价齐升 | 买入 |
| 发酵 | 板块效应，持续上涨 | 持股 |
| 高潮 | 涨停潮，场外抢入 | 逐步卖出 |
| 退潮 | 龙头开板，跟风下跌 | 止损/空仓 |

### 6. 操作建议模板
```
支撑位: XX元
压力位: XX元
止损位: XX元
仓位建议: X-X成
```

## 三、知识体系融合

本分析模式融合了：

1. **曹明成庄家理论**
   - 主力建仓信号
   - 拉升/出货识别
   - 筹码分布

2. **热点题材实战指南**
   - 板块热点识别
   - 情绪周期判断
   - 龙头战法

3. **技术分析基础**
   - 均线系统
   - 量价关系
   - 趋势判断
