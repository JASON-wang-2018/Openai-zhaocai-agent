# Task Reminder Skill

专为 OpenClaw 设计的人物提醒增强技能，确保定时任务不再遗漏。

## 功能

### 1. 创建提醒
当用户要求设置提醒时，自动：
- 解析时间（"周六上午10点" → 具体时间）
- 创建 cron 任务
- 确认设置成功

### 2. 验证任务
定期检查所有 active 任务状态，确保：
- 任务已正确创建
- 下次执行时间合理
- 发现问题及时报告

### 3. 任务列表
- 列出所有待执行任务
- 显示下次执行时间

## 使用示例

用户说以下话时激活：
- "设置提醒"
- "提醒我"
- "每天早上"
- "定时任务"
- "cron"

## 注意事项

- 时间默认使用 Asia/Shanghai 时区
- 提醒时间要确认是"今天"还是"明天"
- 创建后必须回复用户确认
