# 精益生产方法

## 概述
精益生产(Lean)源于丰田生产系统，目标是消除浪费、持续改进。

---

## 五大原则

### 1. 价值 (Value)
- 从客户角度定义价值
- 识别增值vs非增值

### 2. 价值流 (Value Stream)
- 绘制价值流图
- 识别浪费环节

### 3. 流动 (Flow)
- 连续流动
- 减少批量

### 4. 拉动 (Pull)
- 按订单生产
- 减少库存

### 5. 尽善尽美 (Perfection)
- 持续改进
- 追求零缺陷

---

## 八大浪费 (TIMWOODS)

| 浪费 | 说明 | 家电例子 |
|------|------|----------|
| **T**ransport | 运输 | 物料搬运距离 |
| **I**nventory | 库存 | 原材料堆积 |
| **M**otion | 动作 | 员工多余动作 |
| **W**aiting | 等待 | 设备停机 |
| **O**verproduction | 过度生产 | 预测不准 |
| **O**verprocessing | 过度加工 | 精度过高 |
| **D**efects | 缺陷 | 返工 |
| **S**kills | 技能浪费 | 用人错位 |

---

## 5S现场管理

| 5S | 内容 | 目的 |
|-----|------|------|
| **整理** | 区分要/不要 | 腾出空间 |
| **整顿** | 定位放置 | 快速取用 |
| **清扫** | 清洁环境 | 设备维护 |
| **清洁** | 标准化 | 维持成果 |
| **素养** | 养成习惯 | 持续改进 |

## 看板系统 (Kanban)

### 原理
- 拉动式生产
- 适量库存
- 可视化控制

### 类型
1. **取料看板** - 向上游取料
2. **生产看板** - 指示生产数量
3. **紧急看板** - 应急处理

---

## 快速换模 (SMED)

### 目标
将换型时间缩短到10分钟以内

### 步骤

**1. 区分**
- 内部作业（停机做）
- 外部作业（不停机做）

**2. 转化**
- 内部转外部
- 作业并行

**3. 优化**
- 标准化
- 工具改良

---

## 价值流图 (VSM)

### 符号

```
→→→ 物流
━━━ 信息流
▢ 库存
▽ 供应商
○ 客户
```

### 分析步骤
1. 绘制当前状态图
2. 识别浪费
3. 设计未来状态
4. 实施改进

---

## 家电应用

### 案例：洗衣机装配线优化

**问题**：节拍时间不稳定

**改善**：
1. 识别瓶颈工序
2. 平衡工序
3. 实施一个流
4. 导入看板

**结果**：
- 线速提升30%
- 在制品减少50%
