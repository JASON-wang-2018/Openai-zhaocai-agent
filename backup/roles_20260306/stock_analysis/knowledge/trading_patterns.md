# 主力行为与游资模式知识库

## 一、主力坐庄全流程

### 1.1 五大阶段
```
建仓 → 吸筹 → 试盘 → 洗盘 → 拉升 → 出货 → 砸盘
```

### 1.2 建仓阶段（低位吸筹）
**目标**：低成本获取足够筹码

**手法**：
- **打压式建仓**：制造恐慌，低位吸筹。K线"阴跌-急跌-阴跌"，成交量萎缩
- **反弹式建仓**：反复震荡，收集筹码。形成"双底""三底"形态
- **横盘式建仓**：控制股价在固定区间（振幅5-15%），消磨散户耐心
- **逆势建仓**：大盘暴跌时反向买入

**K线特征**：
- 长期窄幅震荡
- 成交量逐步缩量
- 筹码集中度提升

### 1.3 试盘阶段
**目的**：测试抛压、跟风盘

**特征**：
- 放量上影线
- 短暂突破后回落
- 量能放大但股价不涨

### 1.4 洗盘阶段
**目的**：清理浮筹，提高成本

**手法**：
- **打压洗盘**：快速下跌，恐吓散户
- **横盘洗盘**：长期不涨，消磨耐心
- **拉升洗盘**：边拉边洗，高抛低吸

**识别**：
- 缩量回调
- 均线支撑
- 舍不得卖=还没洗完

### 1.5 拉升阶段
**特征**：
- 放量涨停突破
- 连续大阳线
- 换手率急剧放大
- 带动板块效应

**手法**：
- 快速拉升（游资）
- 震荡上行（主力）

### 1.6 出货阶段
**特征**：
- 放量滞涨
- 高位长上影
- 利好密集发布
- 股东减持

**手法**：
- 涨停板出货
- 震荡出货
- 砸盘出货

### 1.7 砸盘阶段
**目的**：不计成本卖出

**特征**：
- 连续跌停
- 放量下跌
- 恐慌抛售

## 二、游资手法大全

### 2.1 炒股养家
**核心理念**：情绪周期

**核心战法**：
- **核按钮**：次日不及预期直接挂跌停卖
- **情绪周期**：高潮后退潮，退潮后冰点，冰点后修复
- **题材预判**：提前介入预期题材

**口诀**：
- "别人贪婪时我更贪婪，别人恐慌时我更恐慌"
- "买入机会，卖出风险"
- "高手买入龙头，超级高手卖出龙头"

**操作节奏**：
1. 冰点转折试错（轻仓）
2. 确认后加仓（主线）
3. 高潮后撤退

---

### 2.2 赵老哥
**核心理念**：龙头战法

**核心战法**：
- **二板定龙头**：一板看不出，二板确认强度
- **只做龙头**：跟风股不做
- **板块效应**：有跟风股才安全

**口诀**：
- "二板定龙头，一板能看出个毛"
- "新题材出现，果断抛弃老题材"

**操作模式**：
- 打板介入（涨停买入）
- 持股待涨（享受溢价）
- 放量滞涨卖出

---

### 2.3 作手新一
**核心理念**：趋势+情绪

**核心战法**：
- **首板介入**：成本优势
- **分歧买入**：烂板变涨停
- **板块联动**：必须板块有涨停效应

**风格**：
- 敢买高位股
- 持股周期略长
- 注重基本面逻辑

---

### 2.4 北京炒家
**核心理念**：首板模式

**核心战法**：
- **打首板**：不做2板以下
- **回封买入**：炸板后回封
- **量价配合**：放量涨停为佳

**口诀**：
- "首板是风险的起点，也是利润的起点"
- "不做低位板，只做首板"

---

### 2.5 陈小群
**核心理念**：情绪合力+做T

**核心战法**：
- **情绪合力**：买在分歧，卖在一致
- **极致做T**：反复做T降成本
- **主线聚焦**：只做主流龙头

**仓位管理**：
- 试错期：10-20%
- 确认期：50%+
- 决战期：满仓

**风控**：
- 单笔亏损<5%
- 不及预期次日割

---

### 2.6 章盟主（章建平）
**核心理念**：资金为王

**核心战法**：
- **大资金运作**：进出上亿
- **趋势为王**：不做超跌
- **分批建仓**：逐步买入

**风格**：
- 偏好大盘股
- 操作周期较长
- 擅长抄底逃顶

---

### 2.7 徐翔
**核心理念**：基本面+重组

**核心战法**：
- **重组股**：提前埋伏
- **业绩拐点**：困境反转
- **长线布局**

**风格**：
- 低调神秘
- 精准抄底
- 长线持有

## 三、情绪周期

### 3.1 周期阶段
```
冰点 → 修复 → 启动 → 发酵 → 高潮 → 分歧 → 退潮 → 冰点
```

### 3.2 各阶段特征

| 阶段 | 市场特征 | 策略 |
|------|----------|------|
| **冰点** | 涨停少、亏钱效应 | 休息/轻仓试错 |
| **修复** | 跌停减少、有反包 | 尝试买入 |
| **启动** | 出现首个2板 | 确认主线 |
| **发酵** | 板块效应出现 | 积极做多 |
| **高潮** | 板块涨停潮 | 持股待涨 |
| **分歧** | 龙头开板、跟风下跌 | 卖出获利 |
| **退潮** | 亏钱效应扩散 | 坚决空仓 |

### 3.3 龙空龙战法
**定义**：只做龙头，无龙空仓

**核心原则**：
1. **只做绝对龙头**：2-3板确认，有带动性
2. **无龙则空仓**：退潮期坚决休息
3. **买在分歧**：首阴、低吸、烂板
4. **卖在一致**：高潮、加速、缩量

**仓位**：
- 主升期：60%-满仓
- 退潮期：0%

## 四、经典形态

### 4.1 分歧转一致
**形态**：烂板/炸板 → 隔日弱转强
**买入**：分歧日尾盘/次日开盘
**卖出**：一致高潮后

### 4.2 弱转强
**形态**：平开/低开 → 快速拉涨停
**信号**：超预期高开
**买入**：打板确认

### 4.3 加速
**形态**：连续缩量涨停
**策略**：持股不动
**卖点**：量能放大、炸板

### 4.4 反包
**形态**：首阴/跌停 → 次日涨停
**买入**：跌停撬板/低吸
**注意**：只做龙头反包

### 4.5 地天板
**形态**：跌停 → 涨停
**买入**：撬跌停
**风险**：极高

## 五、资金流向

### 5.1 主力资金
- **净流入**：主力买入
- **净流出**：主力卖出

### 5.2 散户资金
- **跟风买入**：往往是高点
- **恐慌卖出**：往往是低点

### 5.3 资金合力
- **游资+机构**：大涨
- **游资对倒**：高位出货

## 六、实战要点

### 6.1 选股
1. **看板块**：是否有板块效应
2. **看龙头**：是否领涨
3. **看量价**：是否健康
4. **看人气**：是否市场焦点

### 6.2 买点
- **突破买**：放量突破前高
- **回调买**：缩量回踩均线
- **分歧买**：烂板/分歧低吸
- **首板买**：打板确认

### 6.3 卖点
- **放量滞涨**：量能放大不涨
- **破位下跌**：跌破重要均线
- **情绪高潮**：板块加速大涨
- **龙头首阴**：断板当天

### 6.4 风控
- **止损**：单笔亏损<5%
- **仓位**：分批建仓
- **空仓**：退潮期休息
- **复盘**：每日总结

---

## 七、识别主力行为

### 7.1 吸筹特征
- 缩量横盘
- 股东户数减少
- 机构持股增加

### 7.2 拉升特征
- 放量涨停
- 突破新高
- 板块联动

### 7.3 出货特征
- 高位放量滞涨
- 利好密集发布
- 大宗交易

### 7.4 洗盘特征
- 缩量回调
- 不破关键均线
- 盘中有拉升

---

## 八、总结

### 核心逻辑
1. **跟庄**：识别主力吸筹，跟随拉升
2. **龙头**：只做市场龙头
3. **周期**：尊重情绪周期
4. **纪律**：严格止损止盈

### 生存之道
- **小亏大赚**：截断亏损，让利润奔跑
- **顺势而为**：不做逆势
- **知行合一**：计划交易，交易计划
- **持续学习**：市场在变，人也要变
