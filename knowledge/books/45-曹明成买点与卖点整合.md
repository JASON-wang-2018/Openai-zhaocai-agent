# 曹明成：买与卖的艺术

> 整合自《一本书看透买点与卖点》+ 《股市庄家》  
> 更新：2026-03-09

---

## 一、庄家坐庄流程（八浪循环）

```
准备 → 建仓 → 试盘 → 整理 → 初升 → 洗盘 → 拉升 → 出货 → 反弹 → 砸盘 → 扫尾
```

**核心三步曲**：建仓 → 拉升 → 出货

---

## 二、建仓阶段（养牛）

### 1. 建仓目的
- 在低位收集足够筹码
- 持仓量决定利润量和控盘程度

### 2. 建仓五个阶段
| 阶段 | 特征 |
|------|------|
| 底底台 | 股价横盘，庄家开始收集 |
| 底 | 股价创新低，引出割肉盘 |
| 二底台 | 稍微拉升，超出底底台 |
| 推升货 | 推升走势，涨幅不大 |
| 第二台 | 清洗获利盘，继续吸筹 |

### 3. 建仓信号（买卖点）

#### K线特征
- 低位横盘，阴阳夹杂
- 吸筹末段：连续阳线伴随温和放量
- 突破平台阳线 = 买入信号

#### 均线特征
- 均线从杂乱逐渐变得清晰
- "均线汇流" = 买入信号

#### 量能特征
- 换手率急剧放大
- "量堆" = 资金参与
- "量柱" = 庄家倒货

#### 背离信号
- 底背离 = 尝试买入

---

## 三、洗盘阶段（训练）

### 1. 洗盘目的
| 目的 | 说明 |
|------|------|
| 吸筹锁筹 | 逼迫散户交出筹码 |
| 降低成本 | 高抛低吸摊低成本 |
| 坚定持股 | 让散户享受盈利 |
| 调整仓位 | 优化持仓结构 |
| 等待时机 | 等待板块配合 |

### 2. 洗盘方式

#### 消息面配合
- 利空消息 → 逼迫散户割肉
- 利好不涨 → 识别庄家意图

#### 技术面配合
- 制造破位恐慌
- 假跌破关键均线
- 诱空后快速拉回

### 3. 洗盘阶段买卖点
- 不破关键支撑 = 买入机会
- 缩量回调 = 逢低买入

---

## 四、拉升阶段（奔腾）

### 1. 拉升条件
- 筹码锁饱（控盘度 50-70%）
- 大势配合

### 2. 拉升时机
- 选择大盘强势时拉升
- 借势而为，四两拨千斤

### 3. 拉升特征
- 连续放量上涨
- 股价脱离成本区
- 振幅加剧

---

## 五、出货阶段

### 1. 出货手法
- 制造假突破
- 高位横盘出货
- 利好掩护出货
- 杀跌式出货

### 2. 出货信号
- 放量不涨
- 高位长上影
- 利好兑现下跌
- 换手率异常放大

## 六、买点与卖点总结

### 买点汇总
| 买点 | 条件 |
|------|------|
| 突破平台 | 放量突破整理平台 |
| 回调支撑 | 回调到均线附近缩量 |
| 底背离 | 指标底背离 |
| 均线金叉 | 5日上穿10日 |
| 放量启动 | 放量突破前期高点 |

### 卖点汇总
| 卖点 | 条件 |
|------|------|
| 放量滞涨 | 放量但不涨 |
| 跌破支撑 | 跌破关键均线 |
| 高位长上影 | 射击之星 |
| 利好兑现 | 利好发布反下跌 |
| 趋势反转 | 均线死叉下行 |

## 七、与双系统模型整合

### 系统一：技术面
- 均线多头排列 → 持股
- 跌破MA10 → 卖出
- 放量突破 → 买入

### 系统二：主力行为
- 建仓缩量 → 买入
- 拉升放量 → 持股
- 出货放量 → 卖出

### 曹氏八线理论
- **10日线法则**：站上10日线持股，跌破10日线持币
- **60日线法则**：股价生命线

---

## 八、实战要点

1. **顺势而为**：跟随庄家节奏
2. **量价配合**：上涨放量、下跌缩量
3. **均线持股**：不破均线不走
4. **纪律止损**：亏损8%必须止损
5. **板块效应**：紧跟热点板块
