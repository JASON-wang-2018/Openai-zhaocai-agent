## 一、曹氏八线理论（核心）

### 1. 10日线战法
- **线上持股**：股价在10日均线上方 → 持有
- **线下持币**：股价在10日均线下方 → 卖出
- **核心**：简单实用，但需结合趋势判断

### 2. 天梯抓主升段
- 股价沿特定均线通道上升，如"天梯"
- 不破均线 → 持股待涨
- 跌破 → 止盈/止损

### 3. 长牛线抓长线黑马
- 依托20日/60日均线震荡盘升
- 每次回调到均线附近都是买点

## 二、庄家运作流程（完整八浪）

```
准备 → 建仓 → 试盘 → 整理 → 初升 → 洗盘 → 主升 → 出货 → 反弹 → 砸盘 → 扫尾
```

### 关键三阶段

| 阶段 | 核心任务 | 散户应对 |
|------|----------|----------|
| **建仓** | 收集筹码 | 观望/小仓位试盘 |
| **洗盘** | 清洗浮筹 | 持有/逢低买入 |
| **拉升** | 拉高股价 | 顺势持有 |
| **出货** | 派发筹码 | 逃顶/清仓 |

---

## 三、建仓阶段识别

### K线特征
- 红绿夹杂，窄幅震荡
- 振幅较小，形成横盘区域
- 吸筹末期：连续阳线+放量

### 均线特征
- 从杂乱到清晰
- 短期均线逐渐收敛

### 量能特征
- 温和放量
- "量价齐升"形态

### 换手率信号
- 换手率急剧放大到15%以上
- 成交量形成"量堆"

---

## 四、洗盘阶段识别

### 洗盘目的
1. 吸筹、锁定筹码
2. 降低庄家持股成本
3. 抬高散户持仓成本
4. 坚定散户持股信心
5. 调整仓位结构
6. 等待时机

### 洗盘方式

| 方式 | 特点 | 买点信号 |
|------|------|----------|
| **打压式** | 快速凶猛，时间短 | 缩量企稳后买入 |
| **横盘式** | 横有多长，竖有多长 | 突破平台时买入 |
| **涨停板式** | 边拉边洗 | 涨停板上车 |
| **冲高回落式** | 突破后快速回落 | 回踩支撑买入 |

### 洗盘幅度
- 底部吸货区：回调到成本区附近
- 拉升期：回调幅度20-30%

---

## 五、拉升阶段识别

### 拉升条件
1. 筹码吸饱（控盘度>50%）
2. 大盘配合
3. 题材配合
4. 热点板块效应

### 拉升特征
- 均线多头排列
- 成交量持续放大
- 涨势迅猛，阳线为主

---

## 六、出货阶段识别

### 出货征兆（5大信号）
1. **目标价位到达** → 涨幅超100%
2. **股价严重高估**
3. **利好消息频传**（掩护出货）
4. **该涨不涨**（形态好但不涨）
5. **放量不涨**（量价背离）

### 出货方式
| 方式 | 特点 |
|------|------|
| **拔高式** | 快速拉高后出货 |
| **平台式** | 横盘整理，缓慢派发 |
| **震荡式** | 高抛低吸 |
| **打压式** | 砸盘出货 |

### 出货K线特征
- 放量滞涨
- 阴阳相间
- 频繁出现"避雷针"

## 七、庄家持仓量计算

### 1. 换手率计算法
```
持仓量 ≈ 换手率 × 13.3%
```

**经验值：**
- 换手率200% → 加快吸筹
- 换手率300% → 庄家吸足，准备拉升

### 2. 控盘程度分类

| 控盘程度 | 活跃度 | 拉升空间 | 跟庄建议 |
|----------|--------|----------|----------|
| <20% | 无行情 | - | 不做 |
| 20-40% | 活跃 | 较小 | 短线 |
| 40-60% | 较好 | 较大 | 中线推荐 |
| >60% | 较差 | 巨大 | 长线/黑马 |

### 3. 筹码分布模型
```
30% 浮动筹码（最活跃）
20% 相对稳定
15% 稳定
15% 较稳定
20% 死亡筹码（不流动）
```

## 九、实战操作纪律

### 曹氏铁律
1. **趋势为王** - 顺势而为
2. **线上持股** - 不破均线不走
3. **止损要坚决** - 错了就认
4. **不贪心** - 到了目标位就卖
5. **跟庄不追高** - 等回调买入

### 仓位管理
- 行情不明：空仓/轻仓（<30%）
- 趋势向上：半仓（50%）
- 主升段：重仓（70%+）
- 顶部信号：清仓
